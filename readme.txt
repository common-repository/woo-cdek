== Description ==
Плагин агрегатор служб доставки для их интеграции в WooCommerce

С помощью нашего плагина, вы сможете создать собственный методы(ы) доставки или интегрироваться через API с некоторыми службами доставки.

### Возможности для создания своего метода доставки ###
С помощью нашего плагина, вы можете создать свой собственный метод доставки,
стоимость доставки формируется на базе условной логики. Вы задаете список условий которые влияют на стоимость доставки для покупателя магазина.
Функции условной логики:
* Налог
* Количество товаров
* Стоимость заказа
* Габариты
* Индекс
* Страна
* Множество методов доставки

### Список служб доставки для интеграции через API: ###
1. [СДЭК](https://cdek.ru)
2. [Пешкарики](https://peshkariki.ru)
3. [Забери-Товар.РФ](https://zaberi-tovar.ru)

С помощью данного плагина, вы экономите время на обработке и передачи информации о заказе в службу доставки.
Каждая служба доставки, доступна в виде отдельного метода доставки, что позволит вам добавить нужную службу доставки только в заданных зонах доставки.
Настройки плагина разделены в рамках каждой службы доставки.

### Возможности работы через API служб доставки: ###
* Расчет стоимости доставки на странице заказа (расчет доставки с помощью запроса API службы доставки)
* Виджет выбора пункта выдачи заказа (на базе Яндекс Карт)
* Сохранение данных доставки в Meta box внутри заказа
* Отправка заказов в личный кабинет службы доставки (Premium)
* Кастомная настройка стоимости доставки, в зависимости от выбранных условий.

### ВАЖНО! ###
Для начала работы плагина, необходимо стать партнером выбранной службы доставки и запросить доступ к API нужной вам службы доставки.

== Installation ==

1. Загрузите файлы плагина в каталог " /wp-content/plugins/plugin-name` или установите плагин непосредственно через экран плагинов WordPress.
2. Активируйте плагин через экран "Плагины" в WordPress
3. Используйте экран Woocommerce->Настройки->Доставка для настройки плагина
